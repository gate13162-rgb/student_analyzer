# student_analyzer
the file or the program basically genertes a mini student grade analyzer for a classroom system that processes student scores, evaluates performance, and generates a summary report.
it first processes the score to 2 decimal places and then it calssify teh scores into grading system as below:
Average	     :Grade
90 and above	: A
75 – 89	      : B
60 – 74	      : C
Below 60	    : F
Then at the end it generates a mini student grade analyzer which evaluates the grades and provide the data for how many students got pass and how many got failed
